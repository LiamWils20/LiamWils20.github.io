About

SANTA NEEDS YOUR HELP!!

Visit his workshop, and begin crafting unique and fun little toys for the children of the world. Build them as efficiently as possible, the more you build the more you help.

Project Backstory

Wrap it up, was a collaborative project created for the Festive Jam 2023



With a theme of "Santa's Little Helper", we designed a game which would take the theme literally. The player would begin in Santa's workshop, where they would create a toy by visiting the necessary stations and complete some short and simple mini-games.

Keyboard Controls

  • WASD → Move
  • Mouse → Look Around
  • E → Interact
  • LMB → Minigame
  • TAB → Switch UI
  • Escape → Toggle Pause
