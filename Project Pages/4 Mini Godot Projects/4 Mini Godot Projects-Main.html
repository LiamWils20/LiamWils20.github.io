Overall About

There are 4 mini projects to this page, which can be explored. All of which were created in Godot, as a form of learning experience utilising the Zenva academy courses. Each game is a unique experience, and covers something different within the Godot engine. The games are all free to play, and can be downloaded here or on itch.io.

Baloon Popper

This project is a simple baloon popping simulator. You can click a baloon and it'll expand ever so slightly, it'll expand a few times before eventually going pop. For each baloown you pop, you get a point. The game has only a few baloons to pop at the moment. However, I plan to add a randomness system which make the game endless and more fun to play.


Controls

  • Mouse LMD → Click to Pop

Sir Loopsalot

This project has no gameplay as of yet. It is a simple loop which generates a number of stars to display. That's it. I do however plan to create a game out of it, a guesser game where you have a certain amount of time to guess how many stars there are. The game will be endless, and the amount of stars will increase as you progress.


Controls

  • LMB - Interact
  • Keyboard - To Type
