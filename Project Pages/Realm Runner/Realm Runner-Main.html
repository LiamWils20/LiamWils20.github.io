About

Realm Runner is a platforming-action-puzzle adventure where you will explore different dimensions to retrieve lost fragments, the source of life across realms. Hurry! You must save this reality from the threat lurking within the dark.

Project Backstory

Realm Runner is a collaborative project within Unity. For more details about persons involved and their contributions, check out the team section below, or the my contributions section.


Realm Runner was created as a university project within a 13 week timeframe. It is no longer in-development, however we may one day return to it if the stars align.

Keyboard Controls

  • WASD - Move
  • Space - Jump
  • Hold RMB - Aim
  • LMB - Shoot (Only While Aiming)
  • Escape - Pause
